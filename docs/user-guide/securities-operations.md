# Поступление и выбытие ценных бумаг

## Назначение

Документ **Поступление / выбытие ЦБ** предназначен для оформления операций покупки и продажи ценных бумаг. Документ ведёт партионный учёт: каждая покупка формирует отдельную партию (лот), а при продаже партии списываются по методу ФИФО (первым пришёл — первым ушёл).

## Создание документа

1. Откройте журнал документов **Поступление / выбытие ЦБ**
2. Нажмите **Создать**
3. Заполните реквизиты:
   - **Вид операции** — выберите «Поступление» (покупка) или «Выбытие» (продажа)
   - **Ценная бумага** — выберите бумагу из справочника
   - **Количество** — количество единиц (поддерживаются дробные значения, до 8 знаков после запятой)
   - **Цена** — цена за единицу
   - **НКД** — накопленный купонный доход по сделке (для облигаций; для акций укажите 0)

4. При необходимости добавьте строки в таблицу **Комиссии**:
   - **Вид комиссии** — Биржевая комиссия, Брокерская комиссия или Прочее
   - **Сумма** — сумма комиссии

5. Нажмите **Провести и закрыть**

На форме документа доступны ссылки на движения по регистрам накопления — можно сразу посмотреть, какие записи были сформированы при проведении.

## Поступление (покупка)

При проведении документа с видом операции «Поступление» система:
- Создаёт новую партию ценных бумаг
- Фиксирует стоимость партии (цена x количество)
- Учитывает НКД и комиссии как составные части стоимости партии

## Выбытие (продажа)

При проведении документа с видом операции «Выбытие» система:
- Списывает ценные бумаги по методу ФИФО — сначала самые ранние партии
- Если одна партия не покрывает нужное количество, списание продолжается из следующих партий
- Рассчитывает финансовый результат: разница между выручкой от продажи (с учётом НКД и комиссий) и балансовой стоимостью списанных партий
- Если в портфеле недостаточно бумаг для списания, система выдаст ошибку с указанием дефицита

## Пример

1. Покупка 100 акций по 50 руб., комиссия брокера 25 руб. → стоимость партии 5000 руб., затраты 25 руб.
2. Покупка 80 акций по 60 руб. → стоимость партии 4800 руб.
3. Продажа 120 акций по 70 руб. → ФИФО спишет 100 из первой партии + 20 из второй

## Ограничения

- Все суммы в одной валюте (мультивалютность не поддерживается)
- Переоценка и начисление НКД по удерживаемым бумагам выполняются отдельными операциями
