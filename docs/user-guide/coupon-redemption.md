# Погашение купонов

## Назначение

Документ **Погашение купонов** оформляет получение купонного дохода по облигациям. При погашении списывается ранее накопленный НКД и фиксируется финансовый результат от купонной выплаты.

## Создание документа

1. Откройте журнал документов **Погашение купонов**
2. Нажмите **Создать**
3. Заполните реквизиты:
   - **Дата** — дата погашения купона
   - **Ценная бумага** — выберите облигацию из справочника
   - **НКД** — ставка купона на одну бумагу (в рублях)
4. Нажмите **Провести и закрыть**

## Как работает погашение

При проведении система:

1. Рассчитывает сумму купона к погашению: **НКД (ставка) x количество бумаг** по каждой партии
2. Списывает НКД из учёта по каждой партии:
   - Сначала списывается **НКД накопленный** (начисленный за период владения)
   - Если накопленного НКД не хватает, списывается **НКД полученный** (уплаченный при покупке)
3. Фиксирует итоговую сумму купона в финансовом результате

## Предварительные условия

- По указанной ценной бумаге должны быть остатки в портфеле
- Сумма купона к погашению не должна превышать доступный НКД (накопленный + полученный) — иначе система выдаст ошибку
- Для корректного учёта рекомендуется предварительно оформить начисление НКД за текущий купонный период

## Пример

1. Куплено 100 облигаций, уплачен НКД при покупке 20 руб.
2. Начислен НКД за период владения: 2 руб. x 100 = 200 руб.
3. Погашение купона: ставка 1.5 руб. x 100 = 150 руб.
4. Списывается: 150 руб. из НКД накопленного (200 - 150 = 50 руб. останется)
5. Финансовый результат: +150 руб.

## Порядок списания НКД

| Очерёдность | Источник | Описание |
|-------------|----------|----------|
| 1 | НКД накопленный | Начисленный за период владения (документом «Начисление НКД») |
| 2 | НКД полученный | Уплаченный при покупке (в документе поступления) |

## Ограничения

- Погашение выполняется по одной ценной бумаге за документ
- Автоматический расчёт суммы купона по параметрам бумаги не поддерживается — ставку НКД нужно указывать вручную
