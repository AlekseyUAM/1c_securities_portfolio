# Отчёты для анализа портфеля ценных бумаг

Три аналитических отчёта для просмотра и анализа данных накопленных в регистрах портфеля ценных бумаг.

---

## Отчет «Котировка ЦБ»

### Назначение

Отчёт предназначен для просмотра исторических котировок ценной бумаги за выбранный период. Отображает дату котировки, вид котировки и цену.

### Как использовать

1. Откройте отчёт **Котировка ЦБ**.
2. Укажите параметры отбора:
   - **Ценная бумага** — ценная бумага, для которой нужно посмотреть котировки (обязательный параметр).
   - **Начало периода** — начальная дата диапазона. По умолчанию — начало текущего года.
   - **Окончание периода** — конечная дата диапазона. По умолчанию — текущая дата.
3. Нажмите **Сформировать**.
4. Отчёт выведет таблицу котировок в хронологическом порядке (по дате).

### Колонки

| Колонка | Описание |
|---------|----------|
| **Период** | Дата котировки |
| **Ценная бумага** | Наименование и реквизиты ценной бумаги |
| **Значение** | Цена за единицу бумаги |

### Примеры использования

- Проверить динамику цены акции за квартал
- Найти котировку на конкретную дату
- Экспортировать историю котировок в файл для анализа

## Отчет «Финансовый результат»

### Назначение

Отчёт показывает финансовый результат (прибыль/убыток) по каждой ценной бумаге за выбранный период. Данные агрегированы в разрезе ценной бумаги с выводом общего итога.

### Как использовать

1. Откройте отчёт **Финансовый результат**.
2. Укажите параметры периода:
   - **Начало периода** — начальная дата периода анализа. По умолчанию — начало текущего года.
   - **Окончание периода** — конечная дата периода анализа. По умолчанию — текущая дата.
3. Нажмите **Сформировать**.
4. Отчёт выведет таблицу с суммами финансовых результатов по каждой ценной бумаге и общий итог.

### Колонки

| Колонка | Описание |
|---------|----------|
| **Ценная бумага** | Наименование и реквизиты ценной бумаги |
| **Сумма** | Финансовый результат (положительное число = прибыль, отрицательное = убыток) |

### Примечания

- Отрицательные значения (убытки) выделяются красным цветом для лучшей наглядности.
- Общий итог показывает суммарный финансовый результат портфеля за период.

### Примеры использования

- Оценить прибыльность каждой ценной бумаги в портфеле
- Найти наиболее убыточные позиции
- Проанализировать результативность портфеля за период

## Отчет «Стоимость ЦБ»

### Назначение

Отчёт показывает остатки портфеля ценных бумаг на конкретную дату. Отображает физическое количество, стоимость в разрезе партий и вычисленную итоговую стоимость с учётом переоценки.

### Как использовать

1. Откройте отчёт **Стоимость ЦБ**.
2. Укажите параметр:
   - **Дата отчета** — дата, на которую нужно посмотреть остатки. По умолчанию — текущая дата.
3. Нажмите **Сформировать**.
4. Отчёт выведет таблицу остатков, структурированную в два уровня: сначала по ценным бумагам, внутри каждой ценной бумаги — по партиям (документам поступления).

### Колонки

| Колонка | Описание |
|---------|----------|
| **Ценная бумага** | Наименование и реквизиты ценной бумаги (уровень группировки) |
| **Партия** | Документ поступления/выбытия, идентифицирует партию бумаг (уровень группировки) |
| **Количество** | Физическое количество бумаг в партии, шт |
| **Стоимость** | Балансовая (историческая) стоимость партии |

### Итоги по группам

Для каждой ценной бумаги отчёт выводит промежуточные итоги (группа-footer) по всем числовым колонкам:
- Общее количество этой ЦБ
- Суммарная стоимость всех партий

### Примечания

- Отчёт показывает остатки **на конец дня** указанной даты.
- Если остатки нулевые (портфель пуст на эту дату), отчёт будет сформирован без ошибок и выведет пустую таблицу.
- Используйте этот отчёт для составления отчётности, аудита портфеля и контроля стоимости.

### Примеры использования

- Получить справедливую стоимость портфеля на дату отчётности
- Сравнить затраты на приобретение с текущей справедливой стоимостью (переоценка)
- Контролировать накопленный купонный доход

---

## Общие возможности

Все три отчёта поддерживают:

- **Фильтрацию** — дополнительные отборы через интерфейс отчёта (сверх встроенных параметров)
- **Сортировку** — переупорядочивание строк по любой колонке
- **Экспорт** — в Excel, текстовый формат, печать
- **Сохранение вариантов** — сохранение пользовательских настроек (отборы, сортировка, оформление) для повторного использования

---

## Предварительные условия

Все отчёты работают только при наличии данных в соответствующих регистрах:

- Отчёт **Котировка ЦБ** требует данных в регистре сведений «Котировки ЦБ» (загруженных через обработку «Загрузка котировок»).
- Отчёт **Финансовый результат** требует данных в регистре накопления «Финансовый результат» (формируемых при проведении документов).
- Отчёт **Стоимость ЦБ** требует данных в регистре накопления «Собственные ЦБ» (остатки портфеля).

Если отчёт выводит пустую таблицу, проверьте:
1. Есть ли данные за указанный период/дату в исходном регистре.
2. Верно ли заданы параметры (ценная бумага, период, дата).
