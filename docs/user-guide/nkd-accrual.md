# Начисление НКД

## Назначение

Документ **Начисление НКД** предназначен для начисления накопленного купонного дохода (НКД) по облигациям, находящимся в портфеле. НКД начисляется пропорционально количеству бумаг в каждой партии.

## Создание документа

1. Откройте журнал документов **Начисление НКД**
2. Нажмите **Создать**
3. Заполните реквизиты:
   - **Дата** — дата начисления
   - **Ценная бумага** — выберите облигацию из справочника
   - **НКД** — ставка НКД на одну бумагу (в рублях)
4. Нажмите **Провести и закрыть**

## Как работает начисление

При проведении система:

1. Запрашивает все партии указанной ценной бумаги с положительным остатком
2. Для каждой партии рассчитывает сумму: **НКД (ставка) x количество бумаг в партии**
3. Создаёт движение прихода с начисленным НКД по каждой партии

Начисленный НКД накапливается в учёте и используется при погашении купонов.

## Предварительные условия

- По указанной ценной бумаге должны быть остатки в портфеле
- Если остатков нет — система выдаст ошибку

## Пример

1. В портфеле две партии облигации: 60 шт. и 40 шт.
2. Ставка НКД = 2 руб. на бумагу
3. Результат: первая партия получит НКД 120 руб. (60 x 2), вторая — 80 руб. (40 x 2)

## Ограничения

- Начисление выполняется по одной ценной бумаге за документ
- Документ не влияет на финансовый результат — НКД учитывается до момента погашения купона
