# Переоценка ценных бумаг

## Назначение

Документ **Переоценка ЦБ** выполняет переоценку (mark-to-market) всех ценных бумаг в портфеле на дату документа. Система сравнивает рыночную стоимость каждой партии (по котировке закрытия) с балансовой стоимостью и фиксирует разницу.

## Создание документа

1. Откройте журнал документов **Переоценка ЦБ**
2. Нажмите **Создать**
3. Укажите **Дату** — дату, на которую выполняется переоценка
4. Нажмите **Провести и закрыть**

Документ не содержит дополнительных реквизитов — переоцениваются все бумаги, по которым есть остатки и котировки.

## Как работает переоценка

При проведении система:

1. Запрашивает остатки по всем ценным бумагам в портфеле (по партиям)
2. Для каждой партии получает котировку закрытия из регистра котировок
3. Вычисляет разницу между рыночной и балансовой стоимостью:
   - **Рыночная стоимость** = котировка x количество в партии
   - **Балансовая стоимость** = стоимость приобретения + накопленная переоценка
   - **Разница** = рыночная стоимость - балансовая стоимость
4. Записывает движение по каждой партии с ненулевой разницей

Переоценка может быть положительной (рыночная цена выросла) или отрицательной (цена упала).

## Предварительные условия

- По ценным бумагам должны быть загружены **котировки закрытия** на дату переоценки (или ранее)
- Если для бумаги нет котировки — она пропускается без ошибки
- Если в портфеле нет остатков — документ проводится без движений

## Пример

1. В портфеле 100 акций, купленных по 50 руб. (стоимость 5 000 руб.)
2. Котировка закрытия — 60 руб.
3. Переоценка: 100 x 60 - 5 000 = +1 000 руб.
4. При следующей переоценке с котировкой 55: 100 x 55 - (5 000 + 1 000) = -500 руб.

## Ограничения

- Переоценка учитывается как нереализованная прибыль/убыток и не влияет на финансовый результат
- Для корректной работы необходимо предварительно загрузить котировки закрытия
